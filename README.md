# WeRecord
AKA **微记录**，是一款本地微信辅助安卓应用程序。支持安卓6.0及以上。
## 目前支持
### v1.0
>+ 导出聊天记录、联系人、解密后的完整数据库
>+ 聊天记录简单数据统计
>+ 增删改查聊天记录
>+ 查询单向好友（仅供参考）
>+ 找回已删除好友
>+ 根据WXID添加好友以及
>+ 批量标记已读、未读
>+ 批量隐藏、显示聊天
>+ 批量删除聊天记录
>+ 更多功能开发中...
